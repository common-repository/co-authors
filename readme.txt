=== Co-Authors ===
Contributors: westonruter
Tags: authors, users
Tested up to: 2.6.2
Requires at least: 2.3
Stable tag: trunk

Allows multiple authors to be assigned to a post. Co-authored posts appear on a co-author's posts page and feed. New template tags list co-authors. Editors may assign co-authors via the 'Post Author(s)' box.

== Description ==

This plugin has been forked and is now being maintained by Automattic as [Co-Authors Plus](http://wordpress.org/plugins/co-authors-plus/). Their fork supercedes this original plugin; please use it instead.
